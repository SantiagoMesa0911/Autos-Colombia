# 🅿️ Sistema de Información - Parqueadero Autos Colombia

Este proyecto es un sistema de información desarrollado para el parqueadero **"Autos Colombia"**, orientado a la gestión de entradas y salidas de vehículos, utilizando una arquitectura basada en servicios (SOA) para el backend en Node.js y una aplicación web frontend en Angular.

---

## 🚀 Tecnologías utilizadas

- **Backend:** Node.js + Express + MongoDB + Mongoose
- **Frontend:** Angular 18+
- **Base de datos:** MongoDB
